# Watermark Tool

专业的批量视频打码工具，支持 GPU 加速，快速高效。

## ✨ 功能特点

- 🎬 **批量处理** - 一次性处理多个视频文件
- 🚀 **GPU 加速** - 支持 NVIDIA、AMD、Intel 显卡加速
- 🎨 **自定义打码** - 灵活选择打码区域和方式
- 💾 **多格式支持** - 支持多种视频输出格式
- 🔄 **断点续传** - 支持暂停和继续处理
- 📊 **实时预览** - 实时查看打码效果
- ⚡ **智能检测** - 自动检测硬件配置，最优性能

## 💻 系统要求

- **操作系统：** Windows 10/11 64位
- **显卡：** （可选）支持 CUDA/OpenCL 的 NVIDIA/AMD/Intel 显卡
- **内存：** 建议 4GB 以上
- **磁盘空间：** 至少 500MB 可用空间

## 📖 快速开始

### 1. 下载安装

下载最新版本的 `watermark-tool-vX.X.X.exe`，双击运行即可。

### 2. 添加视频

点击 "添加视频" 按钮，选择要处理的视频文件（支持多选）。

### 3. 设置打码区域

- 在预览窗口中绘制打码区域
- 可以添加多个区域
- 支持调整区域大小和位置

### 4. 开始处理

点击 "开始处理" 按钮，等待处理完成。

### 5. 查看结果

处理完成后，视频会保存在输出目录中。

## 🎮 快捷键

- `Ctrl+A` - 全选视频
- `Delete` - 删除选中视频
- `Space` - 播放/暂停预览
- `Ctrl+S` - 开始处理
- `Ctrl+E` - 导出设置

## 🔧 高级功能

### GPU 加速

软件会自动检测您的显卡配置，优先使用 GPU 加速：
- **NVIDIA:** 支持 NVENC 编码
- **AMD:** 支持 AMF 编码
- **Intel:** 支持 QSV 编码

### 批量处理

支持同时处理多个视频文件：
- 自动排队处理
- 支持暂停/继续
- 实时显示处理进度

### 自定义输出

- 选择输出视频格式
- 调整视频质量
- 设置输出目录
- 自定义文件命名规则
